Description:
For this project I created a quiz system in Python which changes the difficulty of questions depending on how the user is performing. The program i created starts with easier questions and if the user answers these questions correctly, it moves to more difficult questions. If the user gets answers wrong, it goes back to easier questions again. This makes the quiz slightly adaptive instead of being fixed.

The questions are stored in a CSV file and the user’s scores are saved into another file so they can be viewed later. The program also includes options to view all previous scores, search for a specific user, and sort results. There is also a simple text-based chart to show performance which makes it a bit more interactive.

I chose this project because it allowed me to combine different topics from the module such as functions, loops, file handling and basic data structures. I also tried to include input validation so that the program does not crash if the user enters incorrect data. I faced some difficulties in this project for instance handling file reading and user input was challenging for me.
